01

Networking Basics

⬡
Networking = connecting devices to share data.

Think of roads between houses — data travels the road instead of people.

◇
Two terms to remember:
  • Node: Any device on the network — phone, laptop, printer, server.
  • Communication Link: The path data travels — Wi-Fi waves or physical cable.

02

Devices in a Network

Quick rule: End devices use data. Intermediary devices move data.

◫ End Devices

Where data starts and stops.

  • Desktop / Laptop
  • Mobile Phone / Tablet
  • Printer / Smart TV
  • Server

⇄ Intermediary Devices

Hover each to learn more.

⬡ Switch ›

Connects devices in the same building (LAN). Knows exactly which port to send data to — no wasted traffic.

⟿ Router ›

Connects your local network to the internet — like a post office forwarding letters between cities.

◎ Modem ›

Translates your ISP's signals into a format your router understands — the bridge to the outside world.

03

Network Characteristics

The four pillars of a reliable, scalable, secure network. Hover each to explore.

♡

Fault Tolerance

If one cable breaks, the network reroutes automatically — like GPS finding a detour when the road is closed.

⤢

Scalability

Add 1,000 new devices without slowing existing ones. Good architecture grows without crumbling.

◉

Quality of Service (QoS)

VIP bandwidth for video calls — they don't stutter when someone downloads a huge file simultaneously.

⊛

Security — CIA Triad

Confidentiality (who sees it), Integrity (isn't altered), Availability (always accessible).

05

Protocols — The Rules

⊟

A protocol defines What is communicated, How it's formatted, and When it's sent — like a language both devices agree to speak.

◈ Protocol Elements

  • Encoding: Converting text/video into digital signals.
  • Encapsulation: Wrapping data in envelopes with addressing info.
  • Message Size: Breaking large files into packets.
  • Timing: When to send and how fast — avoiding collisions.

⇶ Delivery Methods

  • Unicast: Sending to exactly one device — a direct message.
  • Multicast: Sending to a selected group — a group chat.
  • Broadcast: Sending to every device — a PA announcement.
